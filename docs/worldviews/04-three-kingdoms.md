# 世界观：三国时期（东汉末-三国）

## 1. 时空范围
- 时间：东汉末年至三国鼎立
- 空间：中原、荆益、江东、边地

## 2. 权力结构与秩序
- 皇权衰落，诸侯割据
- 军政体系高度依附主公与集团
- 门第、军功、谋略共同决定上升路径

## 3. 技术与战争形态
- 冷兵器战争为主，后勤与地理关键
- 谍报、外交、联盟、离间影响战局

## 4. 常见叙事冲突
- 忠义与现实政治冲突
- 家族利益与国家大义冲突
- 谋士理性与武将血性冲突

## 5. 可选角色类型（建议）
- 谋士 / 武将 / 郡县文官 / 游侠 / 医官 / 斥候 / 世家子弟 / 流民领袖

## 6. 角色创建约束
- 价值观、称谓、制度需符合古代语境
- 不引入现代科技与现代组织逻辑

## 7. 开局问题
- 角色依附哪方势力？
- 角色最看重“忠、义、利”哪一项？
- 角色在乱世中的底线是什么？
