# 世界观：五代十国（907-979）

## 1. 时空范围
- 时间：唐末之后至北宋统一前
- 空间：中原政权更替区与南方割据政权

## 2. 权力结构与秩序
- 政权频繁更替，忠诚对象不稳定
- 地方军阀与士族、商帮关系复杂
- 安全感与秩序感普遍不足

## 3. 经济与社会形态
- 战乱与重建并存
- 区域性商业网络仍在发展
- 民间生存智慧与地方秩序并重

## 4. 常见叙事冲突
- 改朝换代中的身份切换
- 忠诚与保命的两难
- 地方利益与中央权威的冲突

## 5. 可选角色类型（建议）
- 节度使幕僚 / 押运商队领队 / 地方医者 / 逃亡文士 / 城防将校 / 情报中间人 / 僧道人士

## 6. 角色创建约束
- 强化“局势不稳”的时代底色
- 角色关系网络必须可解释其生存路径

## 7. 开局问题
- 角色所在城池近期是否易主？
- 角色最依赖的保护伞是谁？
- 角色准备如何在乱局中自保？
