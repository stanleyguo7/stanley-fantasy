# 世界观：当代中国（2020s）

## 1. 时空范围
- 时间：约2018年至今
- 空间：一线/新一线/县城/城乡流动场景

## 2. 社会结构与秩序
- 法律与公共治理体系成熟，数字化治理普及
- 教育、就业、住房、医疗是核心民生议题
- 平台经济与内容生态影响个体机会

## 3. 技术与生活形态
- 移动支付、短视频、电商、AI工具日常化
- 远程协作与弹性办公并存
- 信息密度高，注意力竞争激烈

## 4. 常见叙事冲突
- 职业上升与家庭责任冲突
- 代际观念差异（稳定 vs 自我实现）
- 城市机会与生活成本矛盾

## 5. 可选角色类型（建议）
- 都市白领 / 创业者 / 自媒体创作者 / 医护 / 教培从业者 / 基层公务人员 / 自由职业者 / 新市民

## 6. 角色创建约束
- 超自然元素默认关闭（除非用户自定义）
- 角色行为应符合现实法律与社会逻辑

## 7. 开局问题
- 角色所在城市与居住状态？
- 角色最强的现实压力是什么？
- 角色想改变的生活变量是什么？
