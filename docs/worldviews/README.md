# 世界观设定库（Worldview Library）

> 用于第一阶段：先选世界观；第二阶段：在对应世界观中选角色类型。

## 可选世界观（首批）

### 现实向
1. 当代中国（2020s）
2. 中国80年代（改革开放初期）
3. 中国90年代（市场化加速期）

### 历史向
4. 三国时期（东汉末-三国）
5. 五代十国（907-979）

### 虚构向
6. 射雕英雄传（武侠世界范式）
7. 笑傲江湖（门派政治与江湖秩序）
8. 哈利波特（魔法学校与魔法社会）

## 使用方式（v2流程）
1. 先阅读并选择一个世界观文档
2. 根据文档中的“可选角色类型”做单选/多选
3. 进入角色背景问卷（针对该世界观）
