# 世界观：中国90年代（市场化加速期）

## 1. 时空范围
- 时间：约1990-1999
- 空间：沿海城市、工业城市、县城、南下潮流

## 2. 社会结构与秩序
- 市场经济快速发展，行业重组明显
- 国企改革带来身份与职业转换
- 城乡与地区发展差异显著

## 3. 技术与生活形态
- 传呼机、固定电话、电视媒体主导
- 早期互联网萌芽，信息渠道开始变化
- 贸易、制造、流通行业活跃

## 4. 常见叙事冲突
- 铁饭碗稳定性下降带来的焦虑
- 外出打拼与家庭留守矛盾
- 致富机会与社会风险并存

## 5. 可选角色类型（建议）
- 外贸业务员 / 个体老板 / 国企转岗职工 / 南下打工者 / 记者 / 小城青年 / 司机

## 6. 角色创建约束
- 叙事需体现时代信息与交通限制
- 财富路径与风险路径应成对出现

## 7. 开局问题
- 角色是否经历下岗/转岗？
- 角色的第一桶金来自哪里？
- 角色最怕失去什么？
