# 世界观：射雕英雄传（武侠世界范式）

## 1. 时空范围
- 基于金庸武侠语境（宋金对峙背景）
- 江湖与家国并行叙事

## 2. 秩序结构
- 江湖门派、师承谱系、武学传承重要
- 朝堂与边疆局势影响江湖选择
- 名门正派与邪派并非绝对二元

## 3. 能力体系
- 内功、招式、兵器、轻功、奇门
- 师承与机缘共同决定成长上限

## 4. 常见叙事冲突
- 师门规矩与个人情义冲突
- 家国责任与儿女情长冲突
- 名声与真实动机错位

## 5. 可选角色类型（建议）
- 名门弟子 / 散修游侠 / 门派执事 / 边地义士 / 江湖郎中 / 镖局中人 / 情报探子

## 6. 角色创建约束
- 角色武学成长需有师承或事件支撑
- 江湖身份与行为后果要自洽

## 7. 开局问题
- 角色师承或启蒙来源？
- 角色最想证明什么？
- 角色在“侠义”上的个人定义？
