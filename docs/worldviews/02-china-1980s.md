# 世界观：中国80年代（改革开放初期）

## 1. 时空范围
- 时间：约1978-1989
- 空间：城市单位、工厂、学校、乡镇、沿海开放地区

## 2. 社会结构与秩序
- 计划经济向市场机制过渡
- 单位制仍有强影响力
- 观念逐步开放，个体机会增加

## 3. 技术与生活形态
- 通讯与交通效率较低，信息传播较慢
- 家电开始普及但整体物质相对有限
- 个体工商户与早期市场活力出现

## 4. 常见叙事冲突
- 稳定岗位 vs 下海经商
- 传统观念 vs 新思潮
- 家庭责任 vs 个人理想

## 5. 可选角色类型（建议）
- 国营厂职工 / 个体户 / 中学教师 / 乡镇干部 / 文艺青年 / 技术员 / 返乡青年

## 6. 角色创建约束
- 避免套用当代互联网行为逻辑
- 资源获取需符合当时社会条件

## 7. 开局问题
- 角色是否在“单位体系”内？
- 角色对“下海/改革”的立场？
- 角色最想抓住的时代机会是什么？
