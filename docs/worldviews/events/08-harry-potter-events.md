# 事件库｜哈利波特范式（原著主线+战后延展，60事件）（重写版·时间序）

> 以原著七部曲主线为骨架，后续补入战后制度与社会修复事件，保证世界观连续性。

---

### 事件 01｜哈利进入魔法世界（1991）


- **原设锚点**：原著1-7卷核心主线
- **参考范围**：《哈利·波特》1-7卷主线
- **剧情设定**：哈利收到霍格沃茨录取信，魔法与麻瓜世界边界首次被亲历打破。
- **主要角色**：哈利、海格、麦格、德思礼一家
- **剧情发展**：
  1. 对角巷购物建立魔法社会日常感
  2. 分院前的身份偏见开始显影
  3. 霍格沃茨成为主舞台
  4. 三人组关系萌芽
  5. 校内与校外叙事正式并轨
- **小角色卷入点**：新生、图书馆助理、宿舍级长、商店学徒

### 事件 02｜魔法石争夺与伏地魔回影


- **原设锚点**：原著1-7卷核心主线
- **参考范围**：《哈利·波特》1-7卷主线

- **小角色卷入点**：巡夜学生、禁林护送、学院计分员
