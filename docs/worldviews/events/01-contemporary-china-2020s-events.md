- **主要角色**：返乡青年、县域创业者、基层医生、文化工作者
- **剧情发展**：
  1. 宏观政策或社会风向变化先到达基层
  2. 家庭与职业选择被迫重估
  3. 地方执行出现差异与摩擦
  4. 个体通过迁移、转岗或再学习应对
  5. 阶段秩序重新稳定并遗留新问题
- **小角色卷入点**：县城活动策划、合作社记账、社区健康项目
