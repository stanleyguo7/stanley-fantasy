# 事件库｜三国时期（《三国演义》主线版·50事件·时间序）

> 目标：以《三国演义》叙事骨架构建世界观演化链。  
> 原则：忠于主线历史/演义节点；小角色只作“旁观与卷入”视角，不改写大势。

---

### 事件 01｜黄巾起义爆发（184）


- **原设锚点**：黄巾起义（演义开篇）
- **参考范围**：《三国演义》回目1-3（近似锚定）
- **剧情设定**：张角以“苍天已死，黄天当立”号召民变，州郡动荡，朝廷征调各地义勇。
- **主要角色**：张角、何进、刘备、关羽、张飞。
- **剧情发展**：
  1. 太平道扩散，地方官失控；
  2. 朝廷急令募兵，地方豪强自建武装；
  3. 刘关张桃园结义，投入平乱；
  4. 黄巾主力被逐步压制；
  5. 朝廷暂稳，但军阀化种子已埋下。
- **小角色卷入点**：被征发为民夫、乡勇、粮秣押运者。

## 使用建议
- 若用于角色创建：优先从 01-20（乱世成形）、20-35（三足鼎立）、35-50（后期消耗与终局）选定阶段。  
- 小角色定位建议：军中低阶吏卒、地方文吏、运输与医护、城防与治安执行者。  
- 叙事边界：不篡改主线胜负，只在缝隙中书写个体命运。
