# 世界观：笑傲江湖（门派政治与江湖秩序）

## 1. 时空范围
- 金庸武侠语境中的江湖门派时代
- 以门派权力斗争与个人自由为主轴

## 2. 秩序结构
- 门派内部权力结构复杂
- 正邪标签经常服务于政治立场
- 武学秘籍、掌门继承、联盟关系牵动格局

## 3. 能力体系
- 武学心法与招式体系并重
- 角色强弱与“代价”绑定（伤病、心魔、立场）

## 4. 常见叙事冲突
- 个体自由与组织忠诚冲突
- 师徒伦理与权力斗争冲突
- 名门正派形象与现实手段冲突

## 5. 可选角色类型（建议）
- 门派弟子 / 长老亲传 / 散人剑客 / 情报客 / 叛门者 / 江湖中立者

## 6. 角色创建约束
- 角色立场需随事件可演化，不宜静态标签化
- 关键冲突需体现“名与实不一致”的江湖质感

## 7. 开局问题
- 角色最信任的组织是谁？
- 角色最想摆脱的束缚是什么？
- 角色愿意为自由付出什么代价？
