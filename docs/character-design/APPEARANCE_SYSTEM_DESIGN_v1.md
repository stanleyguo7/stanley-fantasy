# 人物外观形象系统设计（v1）

> 目标：把“人物外观”从一句话描述，拆解成可选择、可复用、可生成素材的结构化系统。

## 1. 设计原则
1. **先选项后描写**：先完成结构化选项，再生成自然语言描述。
2. **世界观一致性**：同一人物外观必须受世界观约束（服饰、材料、审美、功能）。
3. **可视化友好**：每个字段尽量可映射到图片/视频生成提示词。
4. **多阶段输出**：支持“草图版”“标准版”“导演版（镜头化）”。

## 2. 外观字段分层

### L1 基础体征（硬约束）
- 年龄段外观
- 生理性别表达（可选）
- 身高区间
- 体型轮廓
- 肤色与皮肤状态

### L2 五官与发型（识别核心）
- 脸型、眉型、眼型、鼻型、唇型
- 发长、发量、发色、发质、发型结构
- 识别点（痣、疤、酒窝、雀斑、眼神特征）

### L3 姿态与气质（动态识别）
- 常态站姿/坐姿
- 行走节奏
- 手部习惯动作
- 视线习惯
- 气质关键词（2-4个）

### L4 服饰系统（场景化）
- 日常服
- 工作/战斗/仪式服
- 季节替换
- 材料、配色、磨损程度
- 搭配层级（内层/中层/外层）

### L5 配件与随身物（叙事钩子）
- 主配件（1件）
- 次配件（2-3件）
- 随身工具/武器/文具
- 与背景故事关联的纪念物

### L6 状态变化（时间轴）
- 正常状态
- 压力状态（疲惫、受伤、愤怒）
- 特殊节点状态（升职、出征、婚礼、流亡）

## 3. 输出标准
- **结构化档案**：适合问卷、检索、版本迭代
- **文字描述**：200字短描 + 600字长描
- **视觉提示词**：用于生图/分镜
