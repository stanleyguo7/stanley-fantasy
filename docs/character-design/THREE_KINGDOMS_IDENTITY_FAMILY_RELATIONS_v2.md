## 二、身份模板（按类别给出可套用家庭背景与关系）

> 为避免文档过长，v2 采用“类别模板 + 身份特例”结构。

### A类（军事与后勤）模板
- 家庭背景候选：
  - 农户次子/次女，家中田地不足；
  - 军户家庭，代际从军；
  - 战乱迁徙后依附军屯；
  - 配偶或手足已在前线。
- 关系模板候选：
  - 与同乡兵/后勤吏形成互保网络；
  - 与地方仓曹既合作又互疑；
  - 与家中照护者长期分离导致矛盾。

### B类（基层治理）模板
- 家庭背景候选：
  - 小吏之家，识字入役；
  - 宗族旁支，靠文书能力谋生；
  - 流民转编户后进入基层体系。
- 关系模板候选：
  - 与里正、乡绅、军方征发官三方拉扯；
  - 与灾民代表有同情或对立关系；
  - 与亲族在赋税负担上发生冲突。

### C类（平民生产）模板
- 家庭背景候选：
  - 世代耕作/捕鱼/手艺家庭；
  - 因战事失地后改行；
  - 家庭劳动力不足（老人幼儿多）。
- 关系模板候选：
  - 与地租方、税吏、商队长期博弈；
  - 与邻里形成互助借粮关系；
  - 与地方势力有保护费/借贷压力。

### D类（女性角色）模板
- 家庭背景候选：
  - 丈夫从军/战亡后由女性主事；
  - 家传医药、织造、账房技能由女性继承；
  - 婚配跨郡导致双边家族关系复杂。
- 关系模板候选：
  - 与宗族长房围绕财产与抚养权冲突；
  - 与县衙小吏形成“公事依赖”；
  - 与同行女性互助网络协同抗风险。

### E类（儿童与少年）模板
- 家庭背景候选：
  - 双亲一方失踪/战亡；
  - 被亲族收养或在行会中学徒；
  - 随商队/渔队迁徙成长。
- 关系模板候选：
  - 与师傅/监护人是依附与成长关系；
  - 与同龄伙伴形成生存结盟；
  - 易卷入大人冲突成为“传话与见证者”。

## 三、身份特例补充（用户点名的平民）

### 19) 农户（重点）
- 家庭背景：
  - 佃田三亩，逢战年常欠租；
  - 家有老母与幼童，青壮稀缺；
  - 与邻村有通婚，互相借种借牛。
- 关系模板：
  - 与地主管事谈租；
  - 与里正因征粮征丁冲突；
  - 与同村形成夜间护田队。

### 20) 渔民（重点）
- 家庭背景：
  - 世代舟居，靠季节水路谋生；
  - 一半收入依赖军方征用渡运；
  - 家中子女熟水性早担活计。
- 关系模板：
  - 与渡口税卡长期摩擦；
  - 与船行管事合作又竞争；
  - 与水军巡检有灰色互惠。

### 23) 小商贩（重点）
- 家庭背景：
  - 挑担走乡，货小利薄但流动性高；
  - 配偶在家兼务农；
  - 孩子可随队当学徒。
- 关系模板：
  - 与城门吏、集市头目日常博弈；
  - 与药铺、酒肆、布贩形成互供网络；
  - 易被军需征调临时供货。

## 四、问卷化选择（v2）

### Step A：身份选择（单选）
- 从 40 个身份中选 1 个主身份。

### Step B：视角选择（单选）
- 女性视角 / 男性视角 / 儿童少年视角 / 不限

### Step C：家庭结构（多选）
- 单亲家庭 / 多代同堂 / 战乱离散 / 学徒寄居 / 小家庭流动

### Step D：生计来源（多选）
- 农耕 / 捕鱼 / 手工业 / 商贩 / 小吏俸给 / 军需相关 / 临时工役

### Step E：关系网络（至少3项）
- 亲族关系、邻里互助、行会关系、官府关系、军方关系、债务关系

### Step F：风险标签（多选）
- 征发风险 / 饥荒风险 / 疫病风险 / 债务风险 / 家庭照护风险 / 身份风险

## 五、边界说明
- 不把女性与儿童角色“工具化”为背景板，允许其拥有主动决策。
- 儿童角色避免不适宜内容，强调生存、学习、传承与见证视角。
- 平民角色优先体现“时代压力下的日常策略”，而非强行英雄化。
