# Project Charter - stanley-fantasy

## 1. 项目愿景
构建一个“虚拟人物资产工厂”：通过互动方式持续生成并完善虚拟人物，使其具备稳定人设、可追踪记忆、可扩展多媒体形象。

## 2. 范围（v0）
- 定义角色目录结构
- 定义角色档案模板（设定、外观、关系、语气、禁忌）
- 定义记忆结构（短期/长期）
- 定义素材存储规范

## 3. 非目标（v0）
- 不直接实现训练/微调
- 不实现复杂前端系统
- 不耦合特定模型供应商

## 4. 成功标准
- 任何新角色可在 10 分钟内完成初始化
- 角色档案完整可读
- 记忆更新过程可追溯（Git 历史）

## 5. 下一步
- 输出系统设计文档（docs/DESIGN.md）并确认
